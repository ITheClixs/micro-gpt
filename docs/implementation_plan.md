# Implementation Plan

## Phase 1: Research Skeleton

Status target: implemented without long training.

- Add importable algorithm modules for backpropagation, CNNs, RNNs, RL, and optimizers.
- Add micro-GPT modules for config, data, model, metrics, dry-run training, and generation.
- Add deterministic tiny configs under `configs/micro_gpt/`.
- Add a Streamlit app that renders live payloads from repository code.
- Add tests for core math and model behavior.

Acceptance:

- `./venv/bin/python -m unittest` passes.
- `./venv/bin/python -m src.micro_gpt.train --config configs/micro_gpt/tiny_debug.json --dry-run` completes on CPU.
- No long training or checkpoint writing is required.

## Phase 2: Research-Grade Experiments

- Add text-corpus loaders for local text, JSONL, and Hugging Face datasets.
- Add checkpointing with metadata: git commit, config hash, seed, dataset identifier, parameter count, token count, hardware, and elapsed time.
- Add evaluation script for validation perplexity and deterministic sample generation.
- Add ablation runner for normalization, positional encoding, MLP type, and optimizer schedule.
- Add app controls for comparing ablation artifacts.

Acceptance:

- Every experiment writes a machine-readable JSONL metrics file.
- Every checkpoint includes config and tokenizer metadata.
- README claims reference committed metrics artifacts.

## Phase 3: Hugging Face and TRL Baselines

- Add dataset validation recipes before any remote job.
- Add documented HF Jobs scripts for SFT, DPO, GRPO, reward modeling, and GGUF conversion.
- Add baseline model cards for SmolLM2 and Qwen3-0.6B comparisons.
- Keep these recipes inactive until the user explicitly approves training.

Acceptance:

- All recipes specify timeout, hardware, Hub push, Trackio logging, and failure modes.
- No remote job launches by default.

## Phase 4: Paper Artifacts

- Add `paper/` with LaTeX or Markdown manuscript scaffolding.
- Export figures from the visualization layer.
- Add reproducibility appendix from configs and metrics.
- Add limitations and negative results.

Acceptance:

- Paper figures are regenerated by scripts.
- Tables cite exact metrics artifacts.
- Claims distinguish completed results from planned experiments.
